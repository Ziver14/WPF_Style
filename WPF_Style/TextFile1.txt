Стиль - это коллекция значений свойств, которые могут применяться к элкменту.
Стили позволяют определить общий набор характеристик форматирования и применять его повсюду в приложении для
обеспечения согласованного вида.
Ключевые свойства, необходимые для создания стилей:
Setters - коллекция объектов которые устанавливают значения для свойств и присоединяют обработчики событий автоматически
	имеют два свойства:
	-Property - указывает свойство к котрому будет применяться данный Setter
	-Value - устанавливает значение 
Tiggers - коллекция объектов которые позволяют автоматически измениять настройки стиля.
	Настройки стиля могут модифицироваться, например, значения какого то другого свойства или при наступлении какого то другого события
Resources - коллекция ресурсов которые должны использоваться со стилями. Например может понадобиться использовать единственный элемент для установки нескольких свойств.
	в таком случае эффетивно создать объект как ресурс и затем использовать егоэтот ресурс в Setters
BasedOn - свойство которое позволяет создавать более спциализированный стиль, наследующий (и дополнительно переопределяющий) параметры другого стиля
- TargetType - свойство которое идентифицирует тип элемента, к которому применяеться данный стиль. Это свойство позволяет создать объекты Setters, влияющие только на определене элементы.
